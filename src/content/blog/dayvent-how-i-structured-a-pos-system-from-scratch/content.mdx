## Glosario

| Término     | Significado                                                                                                                                                                                                                                                                                                           |
| ----------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Sistema POS | Los sistemas de punto de venta o sistemas POS (Point of Sale), son aquellos sistemas cuya finalidad es destinada a cubrir lo que hace una caja registradora en un negocio y sus funciones subyacentes, entre ellas, registrar productos, facturar ventas, hacer seguimiento de las ventas y llevar conteo del dinero. |
| SaaS        | Por software como servicio o Software as a Service (SaaS), se entiende aquellos programas informáticos los cuales brindan un servicio a cambio de un cobro monetario, por ejemplo, Netflix ofrece el servicio exclusivo de streaming a cambio de pagar una suscripción mensual.                                       |

## Planteamiento

Dayvent es un sistema POS para restaurantes el cual está en crecimiento. Actualmente se encuentra haciendo una transición gradual de ser un software privado para un grupo cerrado de personas, ha empezar a ser un SaaS de consumo masivo adoptando clientes nuevos exclusivos hasta estabilizar las funcionalidades en los casos de prueba en escenarios reales.
